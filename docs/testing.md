# Testing Documentation

## Testing Approach

The game is currently tested through manual functional testing.

Each feature is checked against expected behavior.

---

# Test Cases

## Game Initialization

Expected:

- Board appears when page loads
- Cards are randomly arranged

Status:

Passed

## Card Interaction

Expected:

- Cards flip when clicked
- Two cards can be selected
- Matching cards remain visible
- Incorrect matches reset

Status:

Passed

---

## Timer

Expected:

- Timer starts when gameplay begins
- Timer updates correctly
- Timer resets after restart

Status:

Passed

---

## Move Counter

Expected:

- Moves increase after each attempt
- Counter resets correctly

Status:

Passed

## Restart Function

Expected:

- Board regenerates
- Game state resets
- Previous progress disappears

Status:

Passed
