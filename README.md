# Memory Match Game

A browser-based memory card matching game built using HTML, CSS, and JavaScript.

The project focuses on applying software engineering principles such as modular design, separation of concerns, and maintainable code structure.

---

## Features

- Card matching gameplay
- Random card arrangement
- Move tracking
- Timer system
- Match and mismatch detection
- Restart functionality
- Sound effects
- Modular JavaScript architecture

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Git and GitHub

## How To Run

1. Download or clone this repository.
2. Open `index.html` in a web browser.
3. Start playing.

No installation or additional software is required.
